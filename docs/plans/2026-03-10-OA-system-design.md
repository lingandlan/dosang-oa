## 📋 项目概述

**项目名称**: OpenOA 企业办公平台  
**目标**: 为大型团队 (200+) 提供基础 OA + 视频通话能力  
**核心功能**:
- 用户认证与权限管理
- 审批流程 (请假/报销/加班等)
- 考勤打卡 (上班/下班/外勤)
- 公告通知
- 视频会议

## 🔧 微服务划分

| 服务名 | 端口 | 功能 | 技术 |
|--------|------|------|------|
| gateway | 8080 | 路由、鉴权、限流 | Spring Cloud Gateway |
| user-service | 8081 | 用户管理、角色权限 | Spring Boot + MyBatis |
| approval-service | 8082 | 审批流程引擎 | Spring Boot + Flowable |
| attendance-service | 8083 | 考勤打卡、排班 | Spring Boot |
| notice-service | 8084 | 公告发布、推送 | Spring Boot + WebSocket |
| video-service | 8085 | 视频会议管理 | Spring Boot + Jitsi |
| websocket-service | 8086 | 实时消息 | Spring Boot + Netty |

## 💾 数据库设计 (核心表)

**user_service**:
- `sys_user` - 用户表
- `sys_role` - 角色表
- `sys_permission` - 权限表
- `sys_department` - 部门表

**approval_service**:
- `approval_type` - 审批类型 (请假/报销/加班)
- `approval_instance` - 审批实例
- `approval_history` - 审批历史

**attendance_service**:
- `attendance_record` - 考勤记录
- `attendance_rule` - 考勤规则
- `attendance_summary` - 考勤汇总

**notice_service**:
- `notice` - 公告表

## 🌐 API 设计规范

```
GET    /api/v1/users              # 获取用户列表
POST   /api/v1/users              # 创建用户
GET    /api/v1/approvals          # 获取审批列表
POST   /api/v1/approvals          # 提交审批
PUT    /api/v1/approvals/{id}    # 审批操作 (通过/拒绝)
POST   /api/v1/attendance/checkin # 打卡
GET    /api/v1/attendance/today   # 今日考勤
GET    /api/v1/notices            # 公告列表
POST   /api/v1/video/room         # 创建会议室
GET    /api/v1/video/room/{id}   # 获取会议室信息
```
